#	Práctica 1 - Programación Integrativa - Shell Scripting.

##	1.	Programar un script que encuentre todos los ficheros dentro del directorio actual (recursivamente) cuyo nombre cumpla un patrón determinado. El patrón debe proporcionarse como parámetro posicional.

###	1.	Usaremos 'find' para hacer una búsqueda de *ficheros* en el directorio actual:
>	$ find -type f

###	2.	Ahora tenemos que filtrar la salida. Usaremos 'grep'
>	$ find -type f | grep "\.so$"

###	3.	Aislamos la ruta con grep
>	$ find -type f | grep "video [^/] *$"

###	4.	Creamos el script, añadiendo el parámetro.
>	#! /bin/sh\
>	find -type f | grep "$1" 

##	2.	Programar un script que encuentre todos los ficheros dentro del directorio actual (recursivamente) que contengan texto que cumpla un patrón determinado. El patrón debe proporcionarse como parámetro posicional.

###	1.	Vamos a usar 'grep'. El comando será del formato:
>	$ grep "string" -r '.' => esto nos permite buscar de manera recursiva ese patrón en el directorio actual.

###	2.	Ahora tenemos que añadir las opciones para mostrar el nombre del fichero y la línea en la que se encuentra:
>	$ grep -noe "string" -r '.'

###	3.	Creamos el script:
>	#! /bin/sh\
>	grep -noe "$1" -r '.'

###	4.	Ejemplo:
>	$ ./ej2.sh ^.\*foo.\*$

##	3.	Usar el script del paso anterior para encontrar todos los ficheros dentro del directorio actual cuyo contenido incluya la palabra 'código', con o sin tilde.
Para hacer esto, tenemos que ver qué patrón vamos a usar. Lo más cómodo va a ser hacer una cosa así:
>	$ ./ej2.sh c[o,ó]digo
